AEP 01 - Desenvolvimento de uma página Web

O conteúdo da AEP – 01 será ministrado nas aulas até o dia 22/08/2019, sendo que será necessário desenvolver uma página principal com o seu nome e RA, contendo nesta página links para as seguintes páginas:

  • ADS;
  • Música;
  • Filmes;
  • Series;
  • Games;
  • Esportes.

Sendo que nestas páginas deverão conter os conceitos de HTML e CSS vistos em sala. As tags HTML que deverão ser utilizadas são:

  • Títulos (h1, h2, h3, h4, h5, h6);
  • Parágrafo (p);
  • Imagem (img src=””);
  • Link externos (a href=””);
  • Links internos (a href=”#”);
  • Tabelas (table, tr, td);
  • Vídeos (vídeo src=””);
  • Vídeo incorporado do Youtube;
  • Áudio (audio src=””);
  • Lista de itens (ul li);
  • Lista numerada (ol li);
  • Formatação do texto (negrito, itálico, sublinhado, etc);
  • Formatação CSS (cor de fundo, cor de texto, alinhamento do texto, etc).

Na 1ª página a ser desenvolvida, Música, deverá conter informações sobre uma banda a escolha do aluno. A página deverá ter informações sobre os integrantes da banda (com foto individual), foto de todos os integrantes, tabela com os álbuns lançados, alguma informação em lista (pode ser o nome dos integrantes), um áudio de uma música MP3, links para vídeos no Youtube e para outras páginas relacionada a banda, links internos nas páginas, um vídeo incorporado do Youtube, além de informações textuais sobre a banda.

Desenvolver as páginas Filmes e Séries, com informações sobre um filme e uma série a sua escolha. Utilizar informação textual, uma tabela com borda para a quantidade de temporadas e quantidade de episódios, fotos e vídeos. Utilizar estilização em CSS (cor de fundo, fonte, cor do texto, alinhamento, imagem no fundo)

Incluir informação em texto sobre o filme e sobre a série. Utilizar CSS externo, interno e inline. Incluir em cada página (ADS, Música, Filmes, Séries, etc) um link para a página principal.

Para a página Games, adicionar texto, imagens, vídeos e lista sobre um jogo de sua preferência.

Sobre a página ADS, colocar o nome das matérias do curso, indicando as que já cursou, as que está cursando, e as para cursar. Sugestão de inserir as informações em uma tabela (incluir nome do professor, nota e breve descrição da matéria).

Todas as páginas deverão ter:

  • Links internos;
  • Links para a página principal (com exceção da index);
  • Links externos;
  • Informação em texto;
  • Imagens, vídeos, tabelas, listas, etc;
  • Estilização em CSS.

AEP 02 - Desenvolvimento de uma página Web

O conteúdo da AEP – 02 será ministrado nas aulas até o dia 22/08/2019, sendo que será necessário aprimorar o conteúdo das páginas principal e das seguintes páginas:

  • ADS;
  • Música;
  • Filmes;
  • Series;
  • Games;
  • Esportes.

As tags de CSS que deverão ser adicionadas as páginas são:

  • Div;
  • Estilizar com:
    • Padding;
    • Border;
    • Margin;
    • Width;
    • Height.
  • Listas estilizadas:
    • list-style-image: url('');
    • List-style-type: circle, square, none;
    • list-style-position: outside, inside.
  • a:hover.

Um novo link na página principal deve ser adicionado, este link deverá ser para uma página com um formulário em HTML. Este formulário deverá conter inputs dos seguintes tipos:

  • Texto
  • Radio
  • Checkbox
  • Arquivo
  • Password

Utilizar uma função simples de javascript para validar os dados, mostrando um alerta caso o usuário não insira algumas informações.

AEP 03 - Desenvolvimento de uma página Web

O conteúdo da AEP – 03 será ministrado nas aulas até o dia 07/11/2019, sendo que será necessário desenvolver uma nova página principal 2, sendo adicionado um link para a página principal 2 na página principal das AEPs anteriores. Além disto, novas versões das seguintes páginas deverão ser desenvolvidas:

  • Música
  • Filmes
  • Séries
  • Games
  • Esportes

Sendo que nestas páginas deverão conter os conceitos de HTML5 e CSS3 vistos em sala. As tags HTML5 que deverão ser utilizadas são:

  • article
  • aside
  • command
  • details
  • summary
  • figure
  • figcaption
  • footer
  • header
  • mark
  • nav
  • progress
  • section
  • time

A estilização via CSS/CSS3 deverá conter os seguintes itens:

  • Menu com dropdown (no topo ou lateral ou em alguma posição fixa da página)
  • Menu em todas as páginas com mesmo estilo

Para a AEP – 03 deveram ser entregues as páginas de:

  • Filmes
  • Séries

Deverão ser utilizados as tags vistas na aula do dia 07/10/2019 (header, nav, section, article, aside, footer) deixando o conteúdo das páginas de Filmes e Séries com tags do HTML5

No header deverá ser utilizado informação textual e uma imagem

Em nav deverá ser utilizado um menu drop-down, com informações referentes às páginas

A tag section deverá ser composta de pelo menos dois artigos (tag article), sendo também utilizado a tag aside para referenciar o conteúdo interno da página (links internos)

No footer deverá ser utilizado uma imagem e informações sobre o desenvolvedor da página (nome, RA e turma)
